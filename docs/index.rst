Introduction
============

gpi_delamo is a plugin for `gpi <https://github.com/jaredtay/gpi_delamo>`_. It contains some handy widgets for working with the `De-la-mo-v2 <https://github.com/idealab-isu/de-la-mo-v2>`_ python package.

Installation
============

    python setup.py install
